# Ju Pet — App de Gestão

App de gestão da **Ju Pet** (Banho & Tosa Premium · Acupe de Brotas, Salvador-BA).

- **Abrir o app:** https://freudbritho-dotcom.github.io/jupet/
- Single-page HTML/JS, dados salvos no próprio aparelho (localStorage).
- Funções: dashboard, agenda, clientes & pets (múltiplos pets por tutor), alerta de retorno,
  lembretes por WhatsApp, financeiro por período, relatório mensal em PDF e recibo.

`projeto-expansao.html` — projeto estratégico de mudança/crescimento.
